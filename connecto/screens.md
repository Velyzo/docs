### Request Setup

1. **Protocol**: `http` or `https`  
2. **Target**: IP or domain  
3. **Port**: Optional (default if blank)  
4. **Endpoint**: e.g. `/api/test`  
5. **Method**: GET, POST, PUT, DELETE

⏱️ Real-time API calls from your wrist.
